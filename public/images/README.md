# Imágenes de la landing

Estos archivos `.svg` son **placeholders** de marca (azul ACA / amarillo YPF / negro Full)
que reemplazan temporalmente a las fotos reales. Reemplazá cada uno por una foto real
respetando el **mismo nombre de archivo** (podés usar `.jpg`, `.png` o `.webp`; si cambiás
la extensión, actualizá la referencia en el componente correspondiente).

| Archivo | Usado en | Recomendación |
| --- | --- | --- |
| `hero-estacion.svg` | `Hero.astro` (fondo) | Foto panorámica de la estación, ruta o surtidores. Apaisada, mínimo 1600x1000px. |
| `aca-fachada.svg` | `AcaSection.astro` | Foto de la fachada / cartel YPF ACA. Relación 4:3, mínimo 1200x900px. |
| `full-tienda.svg` | `FullSection.astro` (imagen principal) | Interior de la tienda Full. Relación 4:3, mínimo 1000x750px. |
| `full-cafe.svg` | `FullSection.astro` (galería) | Barra de café / bebidas. Cuadrada, mínimo 700x700px. |
| `full-snacks.svg` | `FullSection.astro` (galería) | Góndola de snacks / bebidas frías. Cuadrada, mínimo 700x700px. |
| `og-cover.svg` | `Layout.astro` (Open Graph / redes sociales) | Imagen para compartir en redes. 1200x630px, formato `.jpg` o `.png` recomendado. |
| `socios-ruta.svg` | `src/pages/socios-aca.astro` (hero) | Foto de auto, ruta o familia viajando. Vertical/cuadrada, mínimo 1000x900px. |

El favicon (`favicon.svg`) también puede reemplazarse por el isologo real de Díaz Servicios.
